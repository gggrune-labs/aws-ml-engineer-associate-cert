# AWS Certified Machine Learning Engineer - Associate (MLA-C01)

Study repository for AWS Certified Machine Learning Engineer - Associate certification. This repo contains my learning journey, hands-on projects, and study materials.

## 📋 About This Certification

**Exam Code:** MLA-C01  
**Exam Duration:** 170 minutes  
**Number of Questions:** 65  
**Passing Score:** 720/1000  
**Cost:** $150 USD  
**Validity:** 3 years

### Exam Domains

| Domain | Weight | Topics |
|--------|--------|--------|
| 1. Data Preparation for ML | 28% | S3, Glue, Data Wrangler, Feature Store, data formats |
| 2. ML Model Development | 26% | Algorithms, training jobs, HPO, evaluation metrics |
| 3. Deployment & Orchestration | 22% | Endpoints, Pipelines, MLOps, CI/CD automation |
| 4. Monitoring, Maintenance, Security | 24% | Model Monitor, IAM, VPC, CloudWatch, security |
