## Key Concepts Learned

### SageMaker Basics
- Notebook instances vs SageMaker Studio

- **Amazon S3:** Buckets, prefixes, versioning, lifecycle policies
- **AWS Glue:** Data Catalog, Crawlers, ETL jobs
- **Amazon Athena:** Serverless SQL queries on S3

- **SageMaker Feature Store:** Online/offline feature stores
- **Data formats:** CSV, Parquet, JSON, RecordIO-protobuf
